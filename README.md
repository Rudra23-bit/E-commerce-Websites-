E-Commerce:

Complete eCommerce Website built with React JS & Tailwind CSS

This repository contains a fully functional eCommerce website built from scratch using React JS for frontend development and Tailwind CSS for modern, responsive UI.
The project demonstrates best practices in component-based architecture, state management, and responsive design.

Features:

1.Responsive Web Design: Optimized for desktop and mobile devices

2.Product Listing & Details: View and search products easily

3.Shopping Cart: Add, remove, and manage product quantities

4.Smooth Navigation: Implemented using React Router

5.Reusable Components: Modular design for maintainable and scalable code

6.Modern UI/UX: Tailwind CSS styling for a seamless user experience

Tech Stack:

1.Frontend: React JS, React Router

2.Styling: Tailwind CSS

3.Languages: JavaScript, HTML, CSS
